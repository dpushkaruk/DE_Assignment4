# Firepoint Data Base

## Datasources

We have created 16 tables in OLTP, 2 seed csvs, 1 json file and 1 large csv used for MinIO.

## Airflow

We have created 3 dags: etl pipeline, dbt_build_daily, dbt_build_hourly.

## DBT

We have created 19 tables of Raw and Stg, and 8 analytics tables in Mart.

## Insights

- Bundeswehr is the most effective contractor in our imaginary world.
- FP-9 Missile was ordered the most.
- The most budget was spent on R&D for Mobile Launcher of FP-7 Missile.
- From Anderson, Henson and Jones Aerospace was ordered the most components.
